Subscribe your visitors to MailChimp list via Facebook. Create email subscription via facebook button & get real email and name of your Wordpress visitor

== Description ==
The Social Subscriber plugin is a free tool for quick and easy subscribing users to a web resource mailing via the Facebook social network. The extension easily integrates with MailChimp email marketing service. It helps you to add user to the appropriate mailing lists.

Social Subscriber will save your time to post and customize Facebook subscription button on your website. The installation of the tool will allow users of the web resource to receive useful content in just one click.

= DEMO =

[Demo](https://wow-estore.com/item-preview/social-subscriber/)

= Main features: = 
Make the process of subscribing users easy and enjoyable with the capabilities of Social Subscriber:

* subscription via Facebook;
* integration with the MailChimp email marketing service;
* sending a message to the site administrator;
* redirecting the user to the specified page after the subscription;
* custom button text;
* using a shortcode with attributes to insert a subscription.

= Pro version =

Discover even more features with the Pro version of the Social Subscriber plugin:

* subscription via 4 social networks: Facebook, Google, Twitter, LinkedIn;
* integration with 5 email marketing services: MailChimp, Getresponse, Activecampaign, Aweber, Sendinblue;
* sending a message to the user after the subscription;
* selection of colors and backlight for each button;
* set indents, borders, fonts and icons for all elements;
* add a tag from URL for the mail services and much more;
* add tag from url for email services
